## Installation

Clone the repository and install the dependencies.

```bash
git clone https://github.com/Dunks1980/bay.js-starter.git
cd bay.js-starter
npm install
```

## Usage

### Development

```bash
npm run dev
```

### Production

```bash
npm run build
```

### Preview

```bash
npm run preview
```
