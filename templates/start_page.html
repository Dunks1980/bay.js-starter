An easy to use, lightweight library for web-components.

Web Components.

Bay.js takes a template or file and converts it into a web-component. Share and reuse your components across projects.

Stateful.

Bay.js will update the HTML when state changes and only update the parts of the HTML that need to and also binds events without eval or new Function for stricter security policies.

Syntax.

Bay.js uses standard javascript string literal syntax for binding data to HTML and has a smallest learning curve possible.
